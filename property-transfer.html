🏠 Property Transfer Application

A Property Transfer Application is submitted to update municipal records when ownership of a property changes due to sale, gift, inheritance, partition, or any other lawful transfer. This helps ensure that property records and tax assessments reflect the current owner.

📝 Application Format

To,
The Property Tax Officer,
Municipal Corporation,
Delhi

Subject: Application for Property Transfer

Respected Sir/Madam,

I hereby request the transfer of the property mentioned below in the municipal records. The details are as follows:

  • Name of Previous Owner: ______________________
  • Name of New Owner: ______________________
  • Property Address: ______________________
  • Property ID / UPIC No.: ______________________
  • Transfer Type: Sale / Gift / Inheritance / Partition / Other
  • Date of Transfer: ______________________
  • Contact Number: ______________________

I have attached all supporting documents for verification. Kindly process my request and update the municipal records accordingly.

Thanking You,

Yours faithfully,
______________________ (Applicant Name)
______________________ (Signature)
______________________ (Date)

📌 Guide: How to Apply for Property Transfer

Step 1: Visit the Municipal Portal

Open the official Municipal Corporation website and select the "Property Transfer" service.

Step 2: Fill the Application Form

Enter details of the property, previous owner, new owner, and transfer category.

Step 3: Upload Required Documents

  • Sale Deed / Gift Deed / Will
  • Identity Proof of New Owner
  • Address Proof
  • Property Tax Receipt
  • Death Certificate (for inheritance cases)
  • No Objection Certificate (if applicable)

Step 4: Submit Application

Verify all details carefully and submit the application online.

Step 5: Verification

Municipal authorities will verify the documents and property records before approving the transfer.

Step 6: Record Update

Once approved, the property ownership details will be updated in municipal records.

⚠️ Important Tips

  • Ensure all property taxes are paid before applying.
  • Upload clear copies of all required documents.
  • Check ownership details carefully before submission.
  • Keep the acknowledgement number for tracking purposes.
