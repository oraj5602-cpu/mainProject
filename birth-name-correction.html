📝 Birth Certificate Name Correction Application

A Birth Certificate Name Correction Application is used to request correction of an incorrect name, spelling mistake, typographical error, or other approved amendments in a birth certificate maintained by the Municipal Corporation.

📝 Application Format

To,
The Registrar of Births & Deaths,
Municipal Corporation,
Delhi

Subject: Application for Name Correction in Birth Certificate

Respected Sir/Madam,

I hereby request correction of the name recorded in the birth certificate. The details are as follows:

  • Name as Recorded: ______________________
  • Correct Name: ______________________
  • Birth Certificate Registration Number: ______________________
  • Date of Birth: ______________________
  • Name of Father: ______________________
  • Name of Mother: ______________________
  • Applicant Name: ______________________
  • Mobile Number: ______________________

I have enclosed the required supporting documents for verification. Kindly make the necessary correction and issue an updated birth certificate.

Thanking You,

Yours faithfully,
______________________ (Applicant Name)
______________________ (Signature)
______________________ (Date)

📌 Guide: How to Apply for Birth Certificate Name Correction

Step 1: Visit the Official Portal

Open the Municipal Corporation Birth & Death Registration portal and select the "Birth Certificate Correction" service.

Step 2: Fill the Correction Application

Enter the registration details and provide both the incorrect and corrected name information.

Step 3: Upload Required Documents

  • Original Birth Certificate Copy
  • Identity Proof of Applicant
  • Supporting Proof Showing Correct Name
  • Affidavit (if required)
  • Any Additional Documents Required by the Authority

Step 4: Pay Applicable Fee

Pay the prescribed correction fee, if applicable.

Step 5: Submit the Application

Verify all details carefully and submit the correction request.

Step 6: Verification Process

The Registrar's office will verify the submitted records and supporting documents before approving the correction.

Step 7: Updated Certificate Issuance

Upon approval, the corrected Birth Certificate will be issued and made available for download or collection.

⚠️ Important Tips

  • Ensure all supporting documents match the corrected name.
  • Upload clear and readable document copies.
  • Double-check spelling before submitting the request.
  • Keep the acknowledgement number safe for tracking.
